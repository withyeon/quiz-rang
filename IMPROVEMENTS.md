# 퀴즈랑 개선 제안 사항

## 🎯 우선순위 높음 (즉시 구현 가능)

### 1. 게임 종료 후 결과 화면
- **위치**: `app/game/result/page.tsx`
- **기능**:
  - 최종 순위표 (1등, 2등, 3등 하이라이트)
  - 학생별 통계 (정답률, 평균 응답 시간)
  - 문제별 정답률 차트
  - "다시 하기" 버튼
- **UI**: 카드 형태의 결과 요약 + 애니메이션 효과

### 2. 게임 시작 전 카운트다운
- **위치**: `components/Countdown.tsx`
- **기능**:
  - 선생님이 게임 시작 시 "3, 2, 1, 시작!" 카운트다운
  - 모든 학생 화면에 동시 표시 (Realtime)
  - 카운트다운 사운드 효과
- **UX**: 긴장감 조성, 학생들의 집중도 향상

### 3. 실시간 참가자 수 표시
- **위치**: 로비 화면
- **기능**:
  - "현재 X명 참가 중" 실시간 표시
  - 최소 인원 설정 (예: 2명 이상부터 시작 가능)
  - 참가자 목록 프리뷰
- **UI**: 애니메이션 숫자 + 참가자 아바타 그리드

### 4. 문제별 통계 대시보드
- **위치**: `app/teacher/analytics/page.tsx`
- **기능**:
  - 문제별 정답률 (막대 그래프)
  - 학생별 성취도 (히트맵)
  - 시간대별 참여도
  - 엑셀 다운로드
- **라이브러리**: `recharts` 또는 `chart.js`

## 🚀 우선순위 중간 (중요하지만 시간 소요)

### 5. 게임 히스토리 저장
- **데이터베이스**: `game_sessions` 테이블 추가
- **기능**:
  - 게임 결과 자동 저장
  - 과거 게임 결과 조회
  - 통계 누적 데이터
- **UI**: 캘린더 뷰로 게임 이력 확인

### 6. 커스텀 게임 모드
- **기능**:
  - 시간 제한 모드 (빠르게 답하면 보너스)
  - 팀전 모드 (팀별 점수 합산)
  - 배틀 모드 (1:1 대결)
- **설정**: 선생님 대시보드에서 모드 선택

### 7. 학생별 상세 리포트
- **위치**: `app/teacher/reports/[student_id]/page.tsx`
- **기능**:
  - 개인 성취도 추이 (라인 차트)
  - 약점 분석 (틀린 문제 유형)
  - 학습 추천
- **UI**: 대시보드 스타일의 리포트 페이지

### 8. 문제집 공유 기능
- **기능**:
  - 문제집 공개/비공개 설정
  - 다른 선생님과 문제집 공유
  - 문제집 검색 및 가져오기
- **데이터베이스**: `question_sets` 테이블에 `is_public`, `author_id` 추가

## 💡 우선순위 낮음 (장기 개선)

### 9. 다크 모드
- **구현**: `next-themes` 라이브러리
- **설정**: 사용자 설정에 저장

### 10. 다국어 지원 (i18n)
- **라이브러리**: `next-intl`
- **언어**: 한국어, 영어

### 11. 모바일 앱 (PWA)
- **기능**: 오프라인 모드, 푸시 알림
- **구현**: Next.js PWA 플러그인

### 12. 소셜 로그인
- **제공자**: Google, Kakao
- **라이브러리**: `@supabase/auth-helpers-nextjs`

### 13. 실시간 채팅
- **기능**: 게임 중 학생 간 채팅 (선택적)
- **구현**: Supabase Realtime 채널

### 14. 리더보드 전역 랭킹
- **기능**: 전체 학생 중 상위 랭킹
- **UI**: 시즌별 랭킹 시스템

## 🎨 UI/UX 개선

### 15. 로딩 스켈레톤
- **위치**: 모든 데이터 로딩 화면
- **라이브러리**: `react-loading-skeleton`

### 16. 토스트 알림
- **라이브러리**: `sonner` 또는 `react-hot-toast`
- **사용**: 성공/실패 메시지, 게임 시작 알림

### 17. 접근성 개선
- **키보드 네비게이션**: 모든 버튼, 링크
- **스크린 리더**: ARIA 레이블 추가
- **색상 대비**: WCAG AA 기준 준수

### 18. 성능 최적화
- **이미지 최적화**: `next/image` 사용
- **코드 스플리팅**: 동적 import
- **캐싱**: React Query 또는 SWR 도입

## 📊 데이터 분석

### 19. 고급 통계
- **A/B 테스트**: 문제 유형별 효과 분석
- **학습 곡선**: 시간에 따른 성취도 변화
- **예측 분석**: AI로 학습 성과 예측

### 20. 리포트 자동 생성
- **기능**: 게임 종료 후 자동 리포트 생성
- **형식**: PDF, 이미지, 엑셀
- **공유**: 이메일 자동 발송

## 🛠️ 즉시 구현 가능한 작은 개선사항

1. **게임 시작 전 대기 화면 개선**
   - 참가자 수 표시
   - "게임이 곧 시작됩니다" 메시지
   - 로딩 애니메이션

2. **문제 풀이 시간 표시**
   - 각 문제마다 경과 시간 표시
   - 빠른 정답 보너스 (선택적)

3. **상자 선택 애니메이션 강화**
   - 상자 열리는 애니메이션
   - 보상 표시 시 파티클 효과

4. **순위 변경 알림**
   - 순위가 바뀔 때 토스트 알림
   - "1등을 차지했습니다!" 같은 메시지

5. **게임 종료 축하 화면**
   - 1등 축하 애니메이션
   - "축하합니다!" 메시지
