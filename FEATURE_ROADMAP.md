## 🚀 우선순위 높음 (즉시 구현 권장)

### 1. **학생 관리 시스템** 📚
**왜 필요한가?** 선생님이 학생들을 체계적으로 관리하고 개별 성적을 추적할 수 있어야 합니다.

**기능:**
- 클래스 생성 및 학생 등록
- 학생별 성적 추적 (시간에 따른 성장 그래프)
- 학생별 오답 노트 자동 생성
- 개별 학생 리포트 (PDF 다운로드)
- 출석 체크 기능

**구현 위치:**
- `/app/teacher/students/page.tsx` - 학생 관리 페이지
- `/app/teacher/students/[id]/page.tsx` - 개별 학생 상세 페이지

---

### 2. **숙제 기능** 📝
**왜 필요한가?** ZEP QUIZ처럼 학생들이 집에서도 문제를 풀 수 있어야 합니다.

**기능:**
- 숙제 생성 (문제집 선택, 기한 설정)
- 학생별 숙제 제출 현황
- 자동 채점 및 피드백
- 숙제 완료율 통계
- 부모님 알림 (선택적)

**구현 위치:**
- `/app/teacher/homework/page.tsx` - 숙제 관리
- `/app/student/homework/page.tsx` - 학생 숙제 화면

---

### 3. **문제집 편집 기능** ✏️
**왜 필요한가?** 생성한 문제를 수정하거나 추가/삭제할 수 있어야 합니다.

**기능:**
- 문제 수정 (텍스트, 보기, 정답)
- 문제 추가/삭제
- 문제 순서 변경 (드래그 앤 드롭)
- 문제 난이도 설정
- 문제 태그 추가

**구현 위치:**
- `/app/teacher/sets/[id]/edit/page.tsx` - 문제집 편집 페이지

---

### 4. **즐겨찾기 & 평가 시스템** ⭐
**왜 필요한가?** 좋은 문제집을 저장하고 평가할 수 있어야 자료실이 활성화됩니다.

**기능:**
- 문제집 즐겨찾기
- 별점 평가 (1-5점)
- 댓글 작성
- 추천 문제집 자동 추천
- 내가 만든 문제집 vs 즐겨찾기 분리

**구현 위치:**
- 기존 `/app/teacher/page.tsx`에 즐겨찾기 탭 추가
- `/app/teacher/library/[id]/page.tsx` - 문제집 상세 페이지 (평가, 댓글)

---

### 5. **게임 히스토리 & 리포트** 📊
**왜 필요한가?** 과거 게임 결과를 저장하고 분석할 수 있어야 합니다.

**기능:**
- 게임 세션 자동 저장
- 게임별 상세 리포트 (학생별 정답률, 문제별 난이도)
- 엑셀 다운로드 (Pro 기능)
- 시간대별 성적 추이 그래프
- 클래스 평균 vs 개인 성적 비교

**구현 위치:**
- `/app/teacher/history/page.tsx` - 게임 히스토리
- `/app/teacher/reports/[session_id]/page.tsx` - 상세 리포트

---

## 🎯 우선순위 중간 (중요하지만 시간 소요)

### 6. **팀전 게임 모드** 👥
**왜 필요한가?** 협동 학습과 팀워크를 기를 수 있습니다.

**기능:**
- 팀 자동/수동 배정
- 팀별 점수 합산
- 팀 내 상호작용 (힌트 공유 등)
- 팀 순위표

**구현 위치:**
- `/app/team/page.tsx` - 팀전 게임 페이지
- 게임 모드 선택에 "팀전" 추가

---

### 7. **실시간 채팅/피드백** 💬
**왜 필요한가?** 선생님이 게임 중 학생들에게 피드백을 줄 수 있어야 합니다.

**기능:**
- 선생님 → 학생 일방향 메시지
- 문제별 힌트 제공
- 칭찬 메시지 자동 생성
- 게임 종료 후 피드백 작성

**구현 위치:**
- `components/TeacherChat.tsx` - 선생님 채팅 컴포넌트
- 게임 화면에 채팅 영역 추가

---

### 8. **문제 난이도 자동 조절** 🎚️
**왜 필요한가?** 학생 수준에 맞는 문제를 제공해야 합니다.

**기능:**
- 문제별 난이도 태그 (쉬움/보통/어려움)
- 학생별 맞춤 문제 추천
- 적응형 문제 출제 (정답률에 따라 난이도 조절)
- 학습 곡선 분석

---

### 9. **오답 노트 자동 생성** 📔
**왜 필요한가?** 학생들이 틀린 문제를 복습할 수 있어야 합니다.

**기능:**
- 게임 종료 후 오답 노트 자동 생성
- 오답 문제만 모아서 재시험
- 오답률 높은 문제 집중 학습
- PDF 다운로드

**구현 위치:**
- `/app/student/notes/page.tsx` - 학생 오답 노트
- `/app/teacher/students/[id]/notes/page.tsx` - 선생님 관점 오답 노트

---

### 10. **알림 시스템** 🔔
**왜 필요한가?** 숙제, 게임 시작 등 중요한 이벤트를 알려야 합니다.

**기능:**
- 브라우저 알림 (선택적)
- 이메일 알림 (Pro 기능)
- 게임 시작 알림
- 숙제 마감 알림
- 성적 업데이트 알림

**구현 위치:**
- `components/NotificationCenter.tsx`
- `hooks/useNotifications.ts`
