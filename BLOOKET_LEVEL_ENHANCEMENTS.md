## 💡 구현 예시

### 화면 흔들림 효과
```tsx
const shakeVariants = {
  shake: {
    x: [0, -10, 10, -10, 10, 0],
    y: [0, -5, 5, -5, 5, 0],
    transition: { duration: 0.5 }
  }
}
```

### 글로우 효과
```css
.glow-effect {
  box-shadow: 0 0 20px rgba(255, 215, 0, 0.8),
              0 0 40px rgba(255, 215, 0, 0.6),
              0 0 60px rgba(255, 215, 0, 0.4);
  animation: pulse-glow 2s infinite;
}
```

### 점수 카운트업
```tsx
<AnimatedNumber 
  value={score} 
  duration={1}
  format={(n) => n.toLocaleString()}
/>
```
